# telegram-wiki-bot

## How to run bot

 - make virtualenv:

 	`virtualenv {envname} --python=python3`

 - activate it:

 	`source {envname}/bin/activate`

 - install requirements:

 	`pip install -r req.txt`

 - run bot:

 	`python bot.py`

 ## Bot usage

- Open Telengram and find 'WikiHelper'

- Input `/start` or `/help`

- Enjoy!
